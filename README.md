# Protege Marca

Site demonstrativo para consultoria de registro de marcas e patentes no Brasil.

O site utiliza navegação moderna, seção de serviços, blog e contato via WhatsApp.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- [Font Awesome](https://fontawesome.com/) para ícones

## Estrutura do Site

- `index.html` – Página inicial
- `empresa.html` – Quem somos
- `contato.html` – Fale conosco
- `servicos/` – Páginas de serviços (Registro de Marca, Patente, Direito Autoral, Desenhos Industriais, Registro de Software)
- `blog/` – Artigos sobre marcas e patentes
- `css/style.css` – Estilo do site
- `js/script.js` – Scripts interativos

## Deploy

O site está hospedado em: [https://protege-marca.vercel.app](https://protege-marca.vercel.app)
